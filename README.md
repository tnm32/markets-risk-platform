# Markets & Risk Analytics Platform

A six-page Python/Streamlit platform covering live market data, portfolio 
analytics, Markowitz portfolio optimisation, Value-at-Risk backtesting, 
macro regime classification, and Black-Scholes options pricing.

**Live demo:** https://marketsriskplatform.streamlit.app  
**Built with:** Python, Streamlit, pandas, NumPy, SciPy, Plotly, yfinance, FRED API
